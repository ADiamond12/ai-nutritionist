# AI Nutritionist Demo Storyboard

Use this storyboard for a short public-safe walkthrough of the benchmark project.

## 90-Second Reviewer Flow

1. Run `streamlit run app.py`.
2. Enter a safe sample profile and generate recommendations.
3. Start with the generated daily meal plan, not the empty state.
4. Show the daily nutrition progress panel to explain constraints and targets.
5. Switch to the weekly rotation view to show the planner can create a broader schedule.
6. Open swap alternatives to show the app is not a static report.
7. Close with the boundary: local-first software system, general wellness only, no medical claims.

## What To Say

AI Nutritionist is the portfolio benchmark because it combines data processing, deterministic guardrails, a neural ranker, Streamlit UX, model/data documentation, and tests. It is intentionally framed as general wellness software, not a clinical tool.
