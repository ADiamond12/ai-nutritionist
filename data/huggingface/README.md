# Hugging Face Dataset Export

`food_ranker_items.csv` mirrors the runtime catalog in a simple CSV shape that can be uploaded to a Hugging Face Dataset repository if the project later needs hosted dataset browsing. It includes the processed USDA/FNDDS rows plus the curated Mediterranean/Greek extension.

The export is generated by:

```bash
python scripts/build_food_catalog.py
```

It is derived from USDA FoodData Central FNDDS data and curated USDA-style Mediterranean estimates. It is used here for local neural-ranker training and dataset browsing, not for clinical validation.
